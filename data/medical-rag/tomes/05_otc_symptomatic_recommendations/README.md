# Tomo 05 — OTC y Recomendaciones Sintomáticas

## Propósito

Conocimiento sobre medicamentos de venta libre y recomendaciones sintomáticas, con catálogo médico aprobado.

## Alcance clínico

- OTC aprobados por jurisdicción.
- Contraindicaciones.
- Interacciones seguras.
- Límites de duración y dosis.

## Exclusiones

- Medicamentos controlados.
- Prescripción médica.
- Uso en casos complejos sin supervisión.
