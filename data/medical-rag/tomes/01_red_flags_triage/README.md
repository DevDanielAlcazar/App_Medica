# Tomo 01 — Red Flags y Triage

## Propósito

Proveer conocimiento estructurado sobre signos de alarma clínica que requieren derivación inmediata o evaluación médica presencial.

## Alcance clínico

- Signos de alarma por sistema.
- Criterios de urgencia.
- Derivación a emergencias.
- Preguntas de triage inicial.

## Fuentes esperadas

- WHO guidelines sobre urgencias.
- CDC emergency signs.
- Guías oficiales de triage hospitalario.

## Exclusiones

- No incluye diagnósticos definitivos.
- No incluye tratamientos específicos.
- Solo orientación de "cuándo preocuparse".

## Riesgos principales

- Falso negativo: no identificar una red flag real.
- Falso positivo: generar alarma innecesaria.

## Criterios para aprobar staging

- [ ] Catálogo de red flags completado.
- [ ] Sources en `pending_review` solo.
- [ ] Tests de casos críticos definidos.

## Criterios para aprobar producción

- [ ] Revisado por médico clínico.
- [ ] Revisado por legal.
- [ ] Tests de adversarial clínicos pasados.
- [ ] Score de precisión > 95% en casos de urgencia.
