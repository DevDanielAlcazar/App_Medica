# Tomo 06 — Especialidades Frecuentes

## Propósito

Conocimiento sobre síntomas y orientación por especialidad médica frecuente.

## Alcance clínico

- Dermatología.
- Gastroenterología.
- Cardiología.
- Respiratorio.
- Neurología.
- Endocrinología.
- Ginecología.
- Infectología.
- Traumatología.
- Salud mental básica.

## Exclusiones

- Diagnósticos definitivos.
- Tratamientos específicos complejos.
