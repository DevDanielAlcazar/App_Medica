# Maintenance Calendar

## Frecuencia por tipo de tomo

| Tipo | Frecuencia | Responsable |
|------|------------|-------------|
| Red flags / Triage | Trimestral | Medical board |
| Pediatría | Trimestral | Pediatrics board |
| OTC / Sintomático | Trimestral | Medical board |
| Medicina general adulto | Semestral | Medical board |
| Oncología | Trimestral | Oncology board |
| Imagenología | Semestral | Radiology board |
| Especialidades frecuentes | Semestral | Respective boards |
| Regionalización | Anual | Legal/Regional team |

## Disparadores urgentes

- Cambios en guía oficial → revisión inmediata.
- Alerta de seguridad/errores → parche de emergencia.
- Revocación de licencia → desactivación inmediata.
- Error clínico reportado → investigación en 24h.
