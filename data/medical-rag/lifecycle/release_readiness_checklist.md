# Release Readiness Checklist

## Pre-requisitos

- [ ] Tomo en estado `approved_for_staging`.
- [ ] Todos los chunks tienen metadata completa.
- [ ] Licencias validadas.
- [ ] COI revisado.

## Evaluación clínica

- [ ] Tests de casos clínicos pasados.
- [ ] Tests de red flags pasados.
- [ ] Tests pediátricos pasados.
- [ ] Tests OTC pasados.
- [ ] Sin contradicciones con guías oficiales.

## Evaluación legal

- [ ] Licencias compatibles.
- [ ] Atribuciones correctas.
- [ ] No hay contenido protegido sin permiso.
- [ ] Territorios verificados.

## Evaluación RAG

- [ ] Retrieval funciona sin alucinaciones.
- [ ] Filtros metadata funcionan.
- [ ] Performance aceptable.
- [ ] Tests de seguridad pasados.

## Release

- [ ] CHANGELOG actualizado.
- [ ] Versión aprobada.
- [ ] Deploy a staging verificado.
- [ ] Aprobación médica.
- [ ] Aprobación legal.
